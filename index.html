LectuRate by mkrischer

Realtime Lecture Voting and Asking via public channels like twitter, xmmp, email, etc.

LectuRate - Echtzeitbewertungen von Vortragsveranstaltungen Seminare und Vorträge werden in immer vielfältigeren Umgebungen durchgeführt. Bislang dominiert die Anwesenheit lokaler Zuhörer, doch zeigt der Zukunftstrend immer mehr auf eine gemischte Umgebung mit lokalen und entfernten Zuhörern oder sogar einem Publikum nur aus entfernten Personen. Während die Technik zur Übertragung der seminaristischen Veranstaltungen in Echtzeit bereits in einer sehr ausgereiften Form zur Verfügung steht, ist es bislang für die entfernten Zuhörer nur eingeschränkt möglich, in die Veranstaltung mit einer Rückmeldung einzugreifen. LectuRate richtet sich an die Vortragenden, die eine Echtzeitrückmeldung zu Ihren Vorträgen ermöglichen möchten und an Zuhörer von Vorträgen, denen es ermöglicht wird, mittels verschiedener Kommunikationskanäle mit der dozierenden Person zu interagieren.

Dependencies

Java 1.5, Jabber account for every Client, one accounts for every service the server uses

Install

Client: Just run with java -jar LecturateClient.jar
